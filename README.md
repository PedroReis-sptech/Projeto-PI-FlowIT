# FlowIt: PontoDeVista

## 📖 Sobre o projeto!
##### O PontoDeVista(PDV) é um sistema de monitoramento IoT que utiliza sensores ultrassônicos para captar o fluxo de pessoas em corredores de lojas físicas. Os dados coletados são armazenados em um servidor de banco de dados e apresentados por meio de dashboards interativos, permitindo a análise do comportamento dos clientes dentro do ponto de venda.
##### A solução possibilita identificar padrões de circulação, áreas de maior e menor movimento e falhas operacionais, auxiliando na otimização do layout, na reposição de produtos e na tomada de decisões estratégicas. Dessa forma, o PDV contribui para a melhoria da experiência do cliente e para o aumento da eficiência e das vendas no varejo.
